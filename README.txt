# Registro de Mantenimiento Mensual - ULTRONA

Esta aplicación permite registrar y respaldar mensualmente las mantenciones realizadas al equipo de orinas ULTRONA.

**Empresa**: Inmunodiagnóstico  
**Registros que se pueden ingresar**:
- Fecha y hora (editable)
- Tipo de mantenimiento (lista autocompletable)
- Nombre del operador (lista autocompletable)

Los datos se guardan automáticamente en un Excel y se respaldan en una carpeta dedicada.
